# DBUI

## Synopsis

DBUI is a very basic ORM comprising a set of QML elements

## Installation

DBUI must be installed as a QML module.

```
$ export PATH=$PATH:~/Qt54//5.4/clang_64/bin
$ git clone <this-repo>
$ cd <this-repo>
$ qmake
$ make
$ sudo make install
... installs ...
```
